# 🔐 Secure Login System (Python)
This is a simple secure login system built using Python.

## Features:
- Username & Password Authentication
- 3 Attempts Limit
- Account Lock System
- Error Handling (try/except)
- Login Logging (file handling)

## Technologies Used:
- Python
- File Handling
- Dictionaries
- Loops & Conditions
